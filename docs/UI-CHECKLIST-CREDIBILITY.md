# Checklist: ความน่าเชื่อถือ (Credibility)

ตรวจตามหลัก 2 ข้อ — CTA ที่ชัดเจน, รูปภาพคุณภาพสูง/สไตล์เดียวกัน

---

## 1. Call to Action (CTA) ที่ชัดเจน

ปุ่มที่ต้องการให้คนคลิกควร **ใช้สีตัดกับพื้นหลัง** และ **วางในตำแหน่งที่เห็นง่าย**

### สถานะปัจจุบัน ✅

| จุด | สถานะ | รายละเอียด |
|-----|--------|------------|
| หน้า Landing – Hero | ✅ | ปุ่มหลัก "เริ่มใช้งาน" ใช้สี primary (ตัดกับพื้นหลัง), ขนาดใหญ่ (size="lg"), มี shadow; ปุ่มรอง "สมัครสมาชิก" เป็น outline วางข้างกัน |
| หน้า Landing – Header | ✅ | "สมัครสมาชิก" ใช้ variant default (สีเต็ม) วางขวาบน เห็นง่าย |
| หน้า Landing – CTA Section | ✅ | พื้นหลังสี primary, ปุ่ม "สมัครสมาชิก" ใช้ variant secondary (สีอ่อนตัดกับพื้นเขียว) + shadow |
| Dashboard – โพสต์สิ่งของ | ✅ | ปุ่มหลัก "โพสต์สิ่งของ" ใช้สี primary อยู่ที่ Navbar |
| การ์ดรายการ – ดูรายละเอียด | ✅ | ปุ่ม "ดูรายละเอียด" ใช้ variant secondary ชัดเจน |

### การปรับที่ทำแล้ว

- **CTA Section (โหมดมืด/สว่าง)**: ปุ่มสมัครใช้สีตัดชัด (เช่น `bg-white text-primary` บนพื้น primary) เพื่อให้ contrast ชัดทุกโหมด
- **ปุ่มหลัก Hero**: เน้นด้วย `ring-2 ring-primary/30` (เมื่อโฟกัส) และคงขนาดใหญ่และ shadow ไว้

---

## 2. รูปภาพคุณภาพสูง / สไตล์เดียวกัน

หลีกเลี่ยงรูป Stock ที่ดูปลอม ใช้ **รูปจริง** หรือ **กราฟิกสไตล์เดียวกัน** เพื่อสร้างภาพจำแบรนด์

### สถานะปัจจุบัน ✅

| จุด | สถานะ | รายละเอียด |
|-----|--------|------------|
| รูปรายการสิ่งของ | ✅ | ใช้ **รูปที่ผู้ใช้อัปโหลด** (Cloudinary) ไม่ใช้รูป Stock |
| ไม่มีรูป (fallback) | ✅ | ใช้ **placeholder แบบแบรนด์** (ไอคอน Package + สี muted) ไม่ใช้รูป Stock หรือรูปสุ่ม |
| โลโก้ / ไอคอน | ✅ | ใช้ `/images/exchange.svg` เดียวกันทั้งเว็บ (Navbar, Footer, Landing) |
| LINE QR, PWA icons | ✅ | ใช้ไฟล์ใน `public/images/` (line-qr.png, icon-192/512) สไตล์เดียวกัน |
| ไม่ใช้ Unsplash/Pexels | ✅ | ไม่มี URL รูป Stock จากแหล่งภายนอก |

### การปรับที่ทำแล้ว

- **Placeholder เมื่อไม่มีรูป**: ใช้ SVG ในโฟลเดอร์แบรนด์ (`/images/placeholder-item.svg`) เป็นกราฟิกกล่อง/ของ สี muted ตรงกับ theme (light/dark) แทน placeholder ทั่วไป

---

## สรุป

- **CTA**: ปุ่มหลักใช้สีตัดกับพื้น วางตำแหน่งเห็นง่าย; ปุ่มใน CTA Section มี contrast ชัด (รวมโหมดมืด)
- **รูปภาพ**: ใช้รูปจริงจากผู้ใช้ + placeholder แบบแบรนด์ + ไอคอน/โลโก้ชุดเดียวกันทั้งไซต์

ไฟล์นี้ใช้เป็นแนวทางตรวจและปรับความน่าเชื่อถือต่อได้
